# 🎯 Project Purpose – Mail Guard AI

---

## 📌 Objective

The goal of this project is to build an intelligent system that can automatically detect whether an email message is **spam or not spam (ham)** using Machine Learning techniques.

---

## 🧠 Problem Statement

Spam emails are a major issue in digital communication. They:

- Waste user time
- Pose security risks (phishing, scams)
- Reduce productivity

This project aims to solve this problem by creating a **smart spam detection system**.

---

## 💡 Solution Approach

We developed a **hybrid system** combining:

### 1️⃣ Machine Learning Model

- TF-IDF for feature extraction
- Logistic Regression for classification

### 2️⃣ Rule-Based Detection

- Custom spam keyword list
- Detects high-risk phrases like:
  - "free money"
  - "urgent"
  - "click here"

---

## 📊 Model Evaluation

The system performance is evaluated using:

- Accuracy
- Precision & Recall
- Confusion Matrix

---

## 🖥️ User Interface

A professional frontend is built using Streamlit:

- Clean UI design
- Dark/Light mode
- Confidence score visualization
- Explainable predictions

---

## 🚀 Key Highlights

- Hybrid AI system (ML + rules)
- Explainable AI (why prediction was made)
- Real-time prediction
- User-friendly interface

---

## 📌 Additional Notes

- This project uses a hybrid approach combining Machine Learning and rule-based filtering.
- The system provides explainable predictions with confidence scores.
- Designed with a user-friendly interface for real-time spam detection.

--

## 🎯 Conclusion

Mail Guard AI demonstrates how machine learning can be applied to solve real-world problems like spam detection, combining accuracy with usability and explainability.
